About AtlasIQ

Enterprise Retrieval-Augmented Generation (RAG)

AtlasIQ is an enterprise-grade Knowledge Search platform engineered for strict factual grounding, zero hallucination policies, and verifiable document citations.

⚙️ 4-Stage RAG Pipeline Architecture

difference 1. Ingestion & Chunking

Parses PDF, DOCX, TXT into semantic chunks with 800 char window and 100 char overlap.

hub 2. Hybrid Retrieval (RRF)

Fuses Dense Vector Search (Qdrant) and Sparse Keyword Search (BM25) via Reciprocal Rank Fusion.

psychology 3. Grounded LLM Generation

Generates precise responses constrained exclusively to context retrieved from your internal documents.

verified 4. Citation Verification

Maps exact page references, document quotes, and relevance scores deterministically without LLM hallucination.

🎮 Infrastructure & GPU Acceleration

  • ✓ PyTorch CUDA: RTX 3050 6GB GPU accelerated embeddings (nomic-embed-text-v1.5)
  • ✓ Database: PostgreSQL 16 + Qdrant Vector Engine
  • ✓ LLM Providers: NVIDIA Build Cloud API & Local Ollama
