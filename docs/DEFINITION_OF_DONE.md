# Definition of Done

A module is NOT complete until:

✅ Unit tests exist

✅ Type hints exist

✅ Logging exists

✅ Docstrings exist

✅ Configuration is externalized

✅ Error handling exists

✅ README updated

✅ API documented

✅ No TODO placeholders

✅ No hardcoded values

✅ Passes linting

✅ Docker still builds

Only then is a feature considered complete.
